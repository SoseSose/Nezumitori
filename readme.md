# Nezumitori
An application that aims to replace mouse clicks by keys. (Windows only)
Similar applications include hunt-n-peck (https://github.com/zsims/hunt-and-peck) and KeyMouse (https://github.com/iscooool/KeyMouse).

## How to execute      
Start by executing dist/nezumitori.exe in the folder.
Ctrl + Shift + L keys are used to read the front window, and keys corresponding to each UI element are displayed.
Clicking on a key executes a click on that UI element.

## Features
Created in Python, using Microsoft UiAutomation or Microsoft Active Accessibility.

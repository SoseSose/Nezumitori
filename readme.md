# Nezumitori
キーによってマウスクリック代替を目指すアプリ。(Windowsのみ)
類似のアプリケーションとして、hunt-n-peck(https://github.com/zsims/hunt-and-peck)やKeyMouse(https://github.com/iscooool/KeyMouse)があります。

# 実行方法   
フォルダ内のdist/nezumitori.exeを実行することで起動。
Ctrl + Shift + Lキーで前面ウィンドウの読み取りをはじめ、各UI要素に対応するキーを表示。
そのキーを押すことでそのUI要素に対してクリックを実行。

# 特徴
Pythonで作成。Microsoft UiAutomation またはMicrosoft Active Accessibilityを使用。
